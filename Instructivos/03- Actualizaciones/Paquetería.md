# Paquetería

#### I.1 - Nativa

• Listar actualizaciones disponibles

~~~
apt list --upgradable
~~~

• Instalar actualizaciones

~~~
sudo apt upgrade -y
~~~

#### I.2 - Flatpak

• Ver actualizaciones disponibles

~~~
flatpak remote-ls --updates
~~~

• Instalar actualizaciones

~~~
sudo apt upgrade -y
~~~

> <p> <p>
